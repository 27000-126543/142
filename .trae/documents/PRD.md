# 大型商业综合体全场景运营管理平台 PRD

## 1. 产品概述

大型商业综合体全场景运营管理平台是一套面向购物中心、商业综合体的一站式智慧运营解决方案。平台整合招商、运营、财务、物业、高管决策五大核心业务模块，实现从品牌入驻到日常运营再到数据分析的全流程数字化管理。目标是提升运营效率、降低管理成本、增强决策精准度。

- 核心价值：打通招商-运营-财务-物业-决策数据孤岛，实现全场景智能协同
- 目标用户：招商部、运营部、财务部、物业部、总经理/高管层

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 招商部 | 账号密码登录 | 入驻申请审核、铺位智能推荐、电子合同管理 |
| 运营部 | 账号密码登录 | 促销活动创建、活动预算推荐、销售数据统计 |
| 财务部 | 账号密码登录 | 租金账单生成、滞纳金管理、门禁锁定解锁 |
| 物业部 | 账号密码登录 | 报修工单处理、智能派单、维修评分 |
| 总经理 | 账号密码登录 | 全览数据大屏、热力图分析、租金系数调整 |

### 2.2 功能模块

1. **登录/首页**：角色选择登录、数据概览仪表盘、消息中心
2. **招商管理**：品牌入驻申请、铺位智能推荐、电子合同签署
3. **运营管理**：促销活动管理、销售排行统计、客流分析
4. **财务管理**：租金账单、滞纳金、门禁管理、积分系统
5. **物业管理**：报修工单、智能派单、扫码评价
6. **总经理视图**：热力图、坪效分析、租金调整、运营报告
7. **系统管理**：客流监控预警、积分商城、消息推送

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深邃商务蓝(#1e3a8a) 搭配 奢华金(#d4af37) 点缀，体现高端商业地产的专业与品质感
- **辅色调**：深空灰(#1f2937) 作为中性色，确保信息层级清晰
- **按钮风格**：圆角8px，hover状态带微阴影，主按钮使用渐变效果
- **字体**：标题使用 Playfair Display 衬线字体彰显高端气质；正文使用 Inter 无衬线字体确保可读性
- **布局风格**：侧边导航 + 顶部面包屑 + 卡片式内容区，数据可视化区域采用玻璃拟态效果
- **图标**：Lucide React 图标库，统一线性图标风格
